This directory contains trial materials for SemEval 2018 Competition 10: Capturing Disriminative Attributes. Contents:

evaluation.py: script that reads res/answer.txt and ref/truth.txt and computes the performance scores and writes them to scores.txt. Takes two command line arguments for locations of res and ref directories.

res/answer.txt is a sample results file.

ref/truth.txt is a corresponding sample ground truth file.

Note that the lines of the answer.txt and truth.txt must match.
